the force

Thank you for your interest in becoming an iJedi! Let's get started.

When reworking this clone please reference the original GitHub (here) or test website (here)*


3 Prerequisites:

1. You have an iRobot Create that is fully integrated with a laptop that has some form of camera
2. You have a Skype account
3. You have access to a personal or external server to host the website


3-Step Training:

1. Begin by cloning the GitHub source code. Change line 35 of usetheforce.html to your own Skype name and then put the website up either on your own server or run it locally.
2. Download the Receiver program onto your laptop HERE. Change line 19 to link to your server.
3. Connect the laptop to the iRobot. Let the laptop sit on top of the iRobot and begin the Skype chat. What your iRobot "sees" should now be displayed via Skype.


You are now an iJedi!
